# Aeroplane Tracker

Система отслеживания самолетов в реальном времени через OpenSky Network API.

## Возможности
1. Получение данных о самолетах в любой стране мира
2. Топ N самолетов по высоте полета
3. Фильтрация по стране регистрации
4. Фильтрация по диапазону высот
5. Сохранение/загрузка данных в JSON
6. Валидация данных и обработка ошибок

# Установка

## Клонирование репозитория
```
git clone https://github.com/v1rtefl-coder/coursework_oop
```
## Установка Poetry
```
pip install poetry
```
## Установка зависимостей
```
poetry install
```

# Запуск
```
python main.py
```
## Использование

1. Получить информацию о самолетах в стране
2. Показать топ N самолетов по высоте
3. Фильтрация по стране регистрации
4. Фильтрация по диапазону высот
5. Сохранить текущие данные в файл
6. Загрузить данные из файла
7. Показать все сохраненные самолеты
8. Очистить сохраненные данные
0. Выход
